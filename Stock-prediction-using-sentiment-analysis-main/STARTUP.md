# Running the Stock Prediction Streamlit App

## Quick Command to Run the App
Copy and paste this in your terminal:
```
cd "c:\Users\dandg\OneDrive\Desktop\BITS Project\Stock-prediction-using-sentiment-analysis-main" ; py -3 -m streamlit run app.py
```

## First Time Setup
Run this command once to install all dependencies:
```bash
pip install -r requirements.txt
```

## Alternative Commands

Simple (if already in project directory):
```bash
streamlit run app.py
```

With Python 3 specified:
```bash
py -3 -m streamlit run app.py
```

With virtual environment:
```bash
myenv\Scripts\python.exe -m streamlit run app.py
```

The app will be available at: http://localhost:8501
